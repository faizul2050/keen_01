#  KeenKeeper

KeenKeeper is a modern personal relationship management tool designed to help users nurture and maintain meaningful connections. It acts as a "personal shelf" for your relationships, ensuring you never lose touch with the people who matter most.

## Technologies Used

- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS & DaisyUI
- **Icons:** React Icons (Io, Bs, Fa, Lu)
- **State Management:** React Context API
- **Notifications:** React Toastify
- **Data Fetching:** Server-side fetching with JSON API integration

##  Key Features

1. **Relationship Tracking & Status:** Easily monitor the health of your connections with color-coded status badges (Overdue, Almost Due, On-track). The system calculates the time since your last contact to keep you accountable.

2. **Interactive Timeline:** Log interactions (Calls, Texts, or Video chats) directly from a friend's profile. These actions are instantly added to a global timeline using Context API, providing a history of your social efforts.

3. **Dynamic Personalized Insights:** View detailed profile pages for each contact, featuring relationship goals, specialized tags (e.g., "Family", "Work", "Startup"), and automated "Next Due Date" calculations to help you stay proactive.
